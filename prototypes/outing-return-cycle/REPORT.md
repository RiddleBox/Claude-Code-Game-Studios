## Prototype Report: Outing-Return Cycle

### Hypothesis
外出-归来循环能够创造基本的"陪伴感"：玩家能接受角色周期性离开并返回的概念，这个循环感觉自然、不打扰，又能创造期待感。

### Approach
**构建内容**：
- 基础透明悬浮窗（320×320，置顶，可拖拽）
- 简化C2逻辑：保底冷却20秒 → 随机出发 → 外出10-45秒 → 归来反馈
- 时间加速模拟：1秒 = 1分钟
- 状态可视化：在家/外出中/冷却中
- 手动控制按钮（测试用）

**实现时间**：约2小时
**技术栈**：Godot 4.6.1 + GDScript
**简化假设**：
1. 跳过点击穿透等高级窗口特性
2. 用文本反馈代替完整叙事碎片
3. 时间加速便于快速测试
4. 状态机简化（只有3个状态）
